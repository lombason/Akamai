    2. disallow -d -r <!--[VAR]TICKET-->$TICKET<!--[ENDVAR]--> <!--[VAR]$TARGETS-->$TARGETS<!--[ENDVAR]-->
  2. Proceed as follows to restore cancelled targets
    1. Suspend the region:
      2. suspendtool --network $NETWORK --region <!--[VAR]$TARGETS-->$TARGETS<!--[ENDVAR]--> --ticket <!--[VAR]TICKET-->$TICKET<!--[ENDVAR]-->
    2. Reinstall the targets:
      3. wax -o --advance <!--[VAR]$TARGETS-->$TARGETS<!--[ENDVAR]--> --ticket <!--[VAR]TICKET-->$TICKET<!--[ENDVAR]-->

      NOTE: --advance will incremental/kernel the target automagically.

  3. After a successful install be sure that everything is unsuspended as well as unstickystopped.
    4. erc <!--[VAR]$TARGETS-->$TARGETS<!--[ENDVAR]--> unstickystop akamai -r <!--[VAR]TICKET-->$TICKET<!--[ENDVAR]-->
    suspendtool --network $NETWORK --region <!--[VAR]$TARGETS-->$TARGETS<!--[ENDVAR]--> --ticket <!--[VAR]TICKET-->$TICKET<!--[ENDVAR]--> -u
  4. If the deconstruction is to take place at a later date, then a new deconstruction request must be submitted.
  5. Set Resolution Abstract: "Deconstruction cancelled; Targets restored". CLOSE THE TICKET
